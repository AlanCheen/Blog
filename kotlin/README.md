# Kotlin



Kotlin 开源、静态类型，同时支持面向对象和函数式编程。



Kotlin 是一种与 Android 兼容的语言，它不仅语法简洁、更具表达性，还具备类型安全和空值安全的特性。它与 Java 语言可以无缝互通



1. 开源

2. 静态类型

3. 支持面向对象

4. 支持函数式编程

   





1. 简洁：大大减少样板代码的数量；
2. 安全：避免空指针异常等整个类的错误；
3. 互操作性（interoperability）：充分利用 JVM、Android 和浏览器的现有库；
4. 工具友好：可用任何 Java IDE 或者使用命令行构建。
